# DevOps Learning Project: Docker Infrastructure

## Description
This project created demonstrates a containerzed web server using some **Nginx** and a monitor tool **Glances** managed by **Docker Compose**

## Technologies used 
* Linux (Ubuntu Server)
* Docker & Docker Compose 
* Git & GitHub
* Something else 

## How to run 
1. Clone the repository 
2. Run 'docker-compose up -d'
